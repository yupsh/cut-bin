# yup-cut

```
NAME:
   cut - remove sections from each line of files

USAGE:
   cut OPTION... [FILE...]

      Print selected parts of lines from each FILE to standard output.
      With no FILE, or when FILE is -, read standard input.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --delimiter value, -d value   use DELIM instead of TAB for field delimiter
   --fields value, -f value      select only these fields (comma-separated list)
   --characters value, -c value  select only these characters (comma-separated list)
   --bytes value, -b value       select only these bytes (comma-separated list)
   --only-delimited, -s          do not print lines not containing delimiters (default: false)
   --help, -h                    show help
```
